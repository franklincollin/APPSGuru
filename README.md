# APPSGuru
Sumber Kode Aplikasi APPS Guru

Untuk menggunakan project ini, pastikan anda mempunyai framework Qt veri > 5.6.1. Project ini adalah project aplikasi Android, namun bisa dijalankan di Desktop. Anda bisa mengkonfigurasi project ini layaknya aplikasi Android dengan AndroidManifest.xml dan build.gradle. Nantinya file-file Android tersebut akan dibuat secara otomatis oleh Qt Creator saat anda mengkonfigurasi project ini untuk Android. Jika anda baru dalam framework Qt (Seperti saya :v), anda mungkin harus melihat dokumentasi Qt terlebih dahulu untuk menjalankan project ini di Android
