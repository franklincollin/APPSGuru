# APPSGuru
Sumber Kode Aplikasi APPS Guru

Untuk menggunakan project ini, isi dari folder header_files, cpp_files, dan qml_files dikeluarkan ke dalam folder utama (folder APPSGuru). Folder icons dibiarkan saja seperti itu. Project ini adalah project aplikasi Android, namun bisa dijalankan di Desktop. Jika anda baru dalam framework Qt (Seperti saya :v), anda mungkin harus melihat dokumentasi Qt terlebih dahulu untuk menjalankan project ini di Android
